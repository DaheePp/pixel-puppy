# 픽셀 강아지 (pixel-puppy) 🐕

방치형 픽셀 강아지 키우기 웹 게임. 데스크톱 펫(모니터 위를 돌아다니는 강아지)의 프로토타입.

- **배포 주소**: https://daheepp.github.io/pixel-puppy/ (GitHub Pages, main 브랜치 root)
- **소유자**: 다희 — 리디(Ridi) PM, 비개발자. Claude와 함께 개발을 배우는 중.

## 소통 방식 (중요)

- 항상 한국어로, 친근한 말투로 대화
- 비개발자이므로 코드 변경 시 개념과 동작 원리를 쉽게 설명
- 어려운 기술 용어는 처음 나올 때 풀어서 설명
- 단계별로 차근차근 진행

## 아키텍처

**의도적으로 단일 파일** (`index.html`)에 HTML + CSS + JS가 모두 들어 있다.
빌드 도구, 프레임워크, 외부 라이브러리 없음. 분리하지 말 것 (배포가 파일 1개 업로드로 끝나는 단순함 유지).

### 핵심 시스템 (index.html 내 섹션 주석 참고)

1. **게임 루프**: `requestAnimationFrame`으로 60fps. 매 프레임 "스탯 감소 → 행동 결정 → 화면 그리기"
2. **상태 머신**: 강아지는 항상 하나의 `state`에 있음
   - 자율 행동: `idle, walk, sit, lie, toToy, toSleep, sleep`
   - 유저 액션: `toBowl, eat`(밥), `chase, carry, playing`(놀이), `walkies`(산책), `come`(부르기)
3. **픽셀 렌더링**: 26×24 그리드에 `P(ctx,x,y,w,h,색)` 헬퍼로 fillRect, SCALE=4 확대 + `image-rendering:pixelated`
4. **견종 시스템**: `BREEDS` 객체가 데이터로 정의 (색상 팔레트, 귀/꼬리 타입, 다리 길이, 털 복슬 여부). 새 견종 = 데이터 한 줄 추가
5. **테마**: 집 3종(`grass/beach/room`) + 산책지 3종(`forest/road/park`). `WALK_DEST`로 매핑. `floorTop()`이 테마별 바닥 시작 높이 결정
6. **2D 이동**: 화면 전체가 바닥. y좌표가 클수록(아래) z-index 높고 스케일 큼 (원근감)
7. **저장**: `localStorage` (`pixelPuppySave_v2` 키). 오프라인 시간만큼 스탯 감소 적용 (최저 5 보호)

### 게임 밸런스 (현재 값)

- 감소: 배고픔·행복 3시간에 100→0 (`DECAY.hunger/happy = 108초당 1`), 에너지 6시간 (216초당 1)
- 잠: 1.8초당 1 회복 (~3분 완충). 에너지 12 이하면 자동으로 쿠션 가서 잠, 85 이상이면 기상
- 보상: 놀이 +35행복 -8에너지 -8배고픔 +15XP / 산책 +30행복 -12에너지 -12배고픔 +20XP / 밥 +50배고픔 +10XP / 쓰다듬기 +4행복 +2XP
- 레벨: 필요 XP = 레벨×50, 무한. 크기 보상은 Lv.13에서 최대 1.5배 (이후 보상 없음 — 개선 예정)

### UX 규칙 (유저가 정한 것들)

- 강아지 말풍선은 강아지의 말만. 조작 안내("터치해보세요" 등)는 패널 하단 `#tip` 문구로
- 식사 중에만 버튼 잠금. 다른 활동(놀이/산책/잠)은 버튼 누르면 `interrupt()`로 정리 후 전환
- 놀이/산책은 유저 인터랙션(공 드래그, 바닥 터치)마다 10초 연장, 무반응 10초면 자동 종료
- 프로그레스바: 식사 남은 시간, 잠잘 때 에너지 회복률, 스탯 부족 시 회복률만 표시
- 배경 장식은 이모지 금지, 전부 픽셀 아트로 직접 그림 (이펙트용 ❤️✨🐾 이모지는 허용)

## 배포

`index.html`을 main에 푸시하면 GitHub Pages가 1~2분 내 자동 반영.

## 로드맵

1. **Electron 데스크톱 펫** (최종 목표): 투명 전체화면 창 + 클릭 통과(`setIgnoreMouseEvents` + forward)로 진짜 바탕화면 위를 돌아다니게. 배경/소품은 숨기고 강아지+미니 UI만
2. 레벨 상한 + 달성 보상 (황금 목줄, 왕관 등 — 논의 중)
3. 기록 유지하면서 견종만 바꾸기 (미용 기능 — 논의 중)
